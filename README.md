# rsstowp

As it stands atm put Keys.xml in your app launch directory.

```xml
<UserData>
  <ReaderAPI>
    <Token></Token>
  </ReaderAPI>
	<Feed>
		<Url></Url>
	</Feed>
	<WordPress>
		<Site></Site>
		<Username></Username>
		<Password></Password>
	</WordPress>
</UserData>
```

Uses http://abrudtkuhl.github.io/WordPressSharp/ .
